# LVR Website Build 2025 – Project Overview

**Goal**
Build a premium, cinematic wedding website for Love, Violeta Rose that feels like a film title sequence: smooth motion, thoughtful typography, and clean structure. The site should be fast, mobile-first, and ready for SEO / ads once polished.

**Tech stack**
- Framework: Next.js 14
- Styling: Tailwind CSS + custom components
- Motion: Framer Motion
- Hosting: Vercel
- Repo: YxungHermes/LVR-HM

**Current priorities**
1. Hero experience + navigation polish.
2. Consultation flow (Book Consultation, form UX, roadmap messaging).
3. Pricing / offerings clarity.
4. Mobile layout and responsiveness.
5. Performance & SEO pass before "public launch".

**Key pages**
- `/` – Home / Hero
- `/consultation`
- `/pricing` (or `/offerings`)
- `/process`
- Any future category pages (e.g. destination weddings, cultural ceremonies, etc.)
